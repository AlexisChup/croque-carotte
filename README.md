# Croque carotte

## This is a school project for the 1st semester


### Installation & Launch
`git clone https://github.com/AlexisChup/croque-carotte.git`
<br>
`cd croque-carotte`
<br>

#### Windows Users
`py main.py`

#### Others
`python3 main.py`
